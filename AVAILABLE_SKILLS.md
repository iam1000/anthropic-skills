# Antigravity Skills 목록 및 기능

| 스킬 이름 (Skill Name) | 주요 기능 (Key Functions) | 사용 예시 |
| :--- | :--- | :--- |
| **`algorithmic-art`** | **알고리즘 아트**<br>- p5.js 등을 활용한 제너레이티브 아트 생성<br>- 코드로 만드는 예술 이미지 | "기하학적인 패턴의 아트워크를 코드로 짜줘" |
| **`brand-guidelines`** | **브랜드 가이드 준수**<br>- 특정 브랜드/회사의 컬러 및 폰트 적용<br>- 디자인 일관성 유지 | "우리 회사 브랜드 컬러로 문서를 꾸며줘" |
| **`canvas-design`** | **캔버스 그래픽 디자인**<br>- 포스터, 소셜 이미지 등 그래픽 요소 생성<br>- 시각적 레이아웃 구성 | "행사 홍보용 포스터 이미지를 디자인해줘" |
| **`doc-coauthoring`** | **문서 공동 작성**<br>- 기획서, 제안서 등의 초안 작성 및 피드백 루프<br>- 구조화된 문서 협업 워크플로우 | "프로젝트 제안서를 같이 써보자" |
| **`docx`** | **Word 문서 작성 및 분석**<br>- 보고서, 계약서 등 포맷팅 된 문서 생성<br>- 표, 목차, 헤더/푸터 설정<br>- 기존 문서 내용 분석 | "주간 보고서 양식으로 워드 파일 만들어줘"<br>"이 문서의 스타일을 변경해줘" |
| **`frontend-design`** | **웹 UI 디자인**<br>- HTML/CSS/React 기반의 모던 웹 UI 생성<br>- 반응형 레이아웃 및 스타일링 | "랜딩 페이지 디자인해줘"<br>"대시보드 UI 코드를 짜줘" |
| **`internal-comms`** | **사내 커뮤니케이션**<br>- 공지사항, 뉴스레터, 회의록 작성<br>- 비즈니스 톤앤매너 적용 | "전사 공지 메일 초안 작성해줘" |
| **`mcp-builder`** | **MCP 서버 제작**<br>- LLM과 외부 시스템 연결을 위한 MCP 서버 코드 생성<br>- Python(FastMCP) / Node.js 지원 | "날씨 API를 연동하는 MCP 서버를 만들어줘" |
| **`pdf`** | **PDF 문서 조작**<br>- PDF 생성 (ReportLab)<br>- 텍스트/이미지 추출 (OCR 아님)<br>- 페이지 병합, 분할, 회전 | "이 보고서를 PDF로 저장해줘"<br>"PDF 파일에서 텍스트만 뽑아줘" |
| **`pptx`** | **파워포인트 생성 및 편집**<br>- 슬라이드 자동 생성 (JS/Python)<br>- 텍스트/이미지 추출<br>- 기존 문서 수정 | "다음 주 발표 자료 초안 만들어줘"<br>"PPT 파일 내용을 요약해줘" |
| **`skill-creator`** | **새로운 스킬 제작**<br>- 사용자 정의 스킬(SKILL.md) 템플릿 생성<br>- 스킬 구조 설계 가이드 | "내 업무에 맞는 자동화 스킬을 새로 만들고 싶어" |
| **`slack-gif-creator`** | **Slack용 GIF 생성**<br>- 메신저에서 사용할 수 있는 움짤 생성<br>- 애니메이션 효과 적용 | "Slack에서 쓸 축하 GIF를 만들어줘" |
| **`theme-factory`** | **디자인 테마 생성**<br>- 문서나 슬라이드에 적용할 색상/폰트 테마 생성<br>- 일관된 스타일 셋 제공 | "새로운 프로젝트를 위한 디자인 테마를 만들어줘" |
| **`web-artifacts-builder`** | **복합 웹 아티팩트 빌드**<br>- 다중 컴포넌트 웹 앱 구조 설계<br>- Shadcn UI 등 라이브러리 활용 | "React로 할 일 관리 앱 전체 구조를 만들어줘" |
| **`webapp-testing`** | **웹 앱 테스트 자동화**<br>- Playwright 기반의 브라우저 테스트<br>- UI 기능 검증 및 스크린샷 캡처 | "로컬 서버 3000포트의 로그인 기능을 테스트해줘" |
| **`xlsx`** | **Excel 데이터 처리**<br>- 대량 데이터 시트 생성<br>- 수식/함수 적용 및 차트 데이터 구성<br>- CSV 변환 및 데이터 정제 | "매출 데이터를 엑셀로 정리해줘"<br>"CSV 파일을 엑셀 차트로 시각화해줘" |
